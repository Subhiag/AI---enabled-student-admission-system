# AI-Enabled Student Admission System

## Project Overview
The AI-Enabled Student Admission System is designed to help educational institutes manage student admissions efficiently using Salesforce CRM and AI capabilities.

The system centralizes student information, course management, enrollment processes, approvals, communication, and reporting.

## Objectives
- Reduce manual administrative work
- Improve decision-making using AI
- Automate admission workflows
- Provide real-time reporting dashboards

## Key Features
- Centralized student management
- Course management
- Automated enrollment and approvals
- Role-based access control
- Real-time dashboards and reports
- AI-powered assistance for counselors and admins

## Technology Used
- Salesforce CRM
- Agentforce AI
- Automation workflows
- Reports and dashboards

## Project Phases
1. Requirement Analysis
2. Salesforce Backend Development
3. UI/UX Customization
4. Data Migration and Testing
5. Deployment and Maintenance
